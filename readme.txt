== Description ==

StoryMapJS by Knight Lab is a free tool to help you tell stories on the web that highlight the locations of a series of events. Storymaps Wordpress plugin turns your Wordpress into a StoryMapJS authoring tool. Categorise StoryMap slides and view Slide categories as their own storymap. Easily embed StoryMaps into posts or pages with shortcodes.

Features

*	Create fantastic looking Storymaps within Wordpress, quickly and easily.
*	Embed Storymaps within Posts and Pages effortlessly!
*	Storymap Slides, Storymap Slide Categories and Storymaps are easily created and managed.
*	Storymap Slide Categories can be viewed as their own Storymap.
*	Storymap Slide Categories have an optional Overview Slide which can be customised.
*	All the features of StoryMapJS including Gigapixel Storymaps

== Installation ==

StoryMaps is extremely easy to install and use.

1. Upload `storymaps` to the `/wp-content/plugins/` directory & activate
1. Upload `advanced custom fields` or `advaned custom fields pro` to the `/wp-content/plugins/` directory & activate
1. Activate StoryMaps
1. Create a few 'Storymap Slides'
1. Create a Storymap and add Storymap Slides to it, order them if required.
1. View Storymaps at its page URL or embed it into your post/page with this shortcode: `[storymap id="example-unqiue" name="basic-storymap-example"]`  (id is a unique div id, name is the storymaps 'slug'). You can also embed a storymap slide-category into a post with the shortcode `[storymap_archive slide_category="slide-category-slug"]`
1. View Gigapixel Storymaps at its page URL or embed it into your post/page with this shortcode: `[gigapixel_storymap id="example-unqiue" name="basic-storymap-example"]`  (id is a unique div id, name is the gigapixel storymaps 'slug')
